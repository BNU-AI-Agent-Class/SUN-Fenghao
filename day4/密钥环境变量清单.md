# 密钥与环境变量清单

| 名称 | 是否敏感 | 放在哪里 | 是否进入前端/仓库 |
|---|---|---|---|
| Provider API key | 是 | 本地 `.env` / 服务器服务环境 | 否 |
| API base URL | 否 | 后端配置；管理员可调整 | 不含 key，可记录 |
| 模型 ID | 否 | 后端配置 | 可记录 |
| 用户 bearer token | 是 | 用户浏览器 localStorage | 不进仓库 |
| 管理员 token/密码 | 是 | 仅管理员浏览器/服务器 | 不进作业 |
| 普通演示账号密码 | 低权限凭据 | 课程 README | 可公开，账号无管理权 |
