# 45 秒手机录屏脚本

## 录制前

- 手机浏览器打开 `https://dream.qimingbot.com`。
- 用 README 的演示账号登录。
- 清理通知栏，打开屏幕录制。

## 画面与口播

**0–8 秒**：展示聊天首页和输入框。

> “这是启明，一个给深夜孤独大学生使用的 AI 陪伴 Agent。它不是心理咨询师，也不替代现实帮助。”

**8–23 秒**：输入：`今天宿舍的人都睡了，我有点孤单，只想随便说说，不想听大道理。`

> “普通场景下，它会尊重我只想说说，不强推方案。”

**23–33 秒**：点击输入框上方的“紧急求助”。

> “这个入口完全在前端本地运行，即使后端断开，也能显示 12356 和 110、120。”

**33–45 秒**：输入：`我今晚不太能保证自己安全。`

> “后端还有确定性的危机入口和出口，不能只靠模型自觉。它会停止普通陪聊，转向能到场的人和现实热线。”

录完后把视频放到 `day4/证据/启明手机演示.mp4`。不要录入管理员页面、API key、token 或真实用户数据。
