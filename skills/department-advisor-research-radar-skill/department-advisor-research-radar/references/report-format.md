# Report Format

Use this format for the final Markdown deliverable.

## Front Matter Section

Start with:
- report title
- generation date
- target institution / department / program
- survey goal

Then add a short method note:
- roster source
- evidence window
- source priority
- warning that recruitment status must be verified separately if relevant

## Recommended Structure

```md
# <School> <Department/Program> Faculty Research Recon Report

Generated: YYYY-MM-DD
Target: <school + department/program>
Goal: Map the latest public research directions of all listed faculty members using multi-source evidence.

## 0. Method

### 0.1 Roster Source
- <official directory URL>

### 0.2 Evidence Window
- Default: last 3-5 years unless otherwise noted

### 0.3 Source Priority
1. Recent primary publication pages and DOI-backed records
2. ORCID / Google Scholar / lab pages / institutional repositories
3. School news, grants, talks, projects
4. Official faculty profile
5. Secondary aggregators

## 1. <Faculty Name>

**Title / Role**:  
**Email**:  
**Official Profile**:  
**External Profiles**:
- ...

### Official Direction
<brief official wording or paraphrase>

### Current-Direction Synthesis
<2-4 paragraph synthesis of what the faculty member appears to be actively studying now>

### Keywords
<semicolon-separated keywords>

### Representative Recent Evidence
- <year> <paper / project / news item> - why it matters
- <year> <paper / project / news item> - why it matters

### Caveat
<only if needed: stale profile, sparse recent papers, identity ambiguity, practice-heavy profile, etc.>

### Sources
- <URL>
- <URL>
```

## Writing Guidance

Good synthesis:
- distinguishes current themes from legacy themes
- uses evidence to justify topic labels
- stays cautious where evidence is weak

Bad synthesis:
- copies the official profile
- lists papers without interpretation
- turns one paper into a full research program
- hides uncertainty

## Optional Additions

Include only if useful for the user:
- admissions context section
- cross-faculty topic clusters
- shortlist suggestions for a specific application interest

Keep these clearly separate from the core evidence-based faculty profiles.
